# Schengen Visa Data Extractor

موقع ويب لاستخراج البيانات من صور الجوازات وتذاكر الطيران وتأكيدات الفنادق، مع ترجمة البيانات إلى الإنجليزية وتصديرها إلى PDF لتأشيرة الشنغن.

## التقنيات المستخدمة

### Frontend
- **React** - مكتبة JavaScript لبناء واجهة المستخدم
- **TypeScript** - لغة برمجة مع دعم الأنواع
- **Tailwind CSS** - إطار عمل CSS للتنسيق

### Backend
- **Node.js** - بيئة تشغيل JavaScript
- **tRPC** - إطار عمل للـ API
- **Drizzle ORM** - للتعامل مع قاعدة البيانات

### قاعدة البيانات
- **MySQL** - قاعدة بيانات علائقية

### الذكاء الاصطناعي
- **Gemini 2.5 Flash** - لاستخراج البيانات من الصور والـ PDF
- **Vision API** - لقراءة النصوص من المستندات

### التخزين
- **S3-compatible Storage** - لتخزين الصور المرفوعة

## المتطلبات

- Node.js 18+
- MySQL Database
- OpenAI API Key (أو مفتاح API متوافق)
- S3 Storage (AWS S3 أو بديل متوافق)

## متغيرات البيئة المطلوبة

```env
DATABASE_URL=mysql://user:password@host:port/database
OPENAI_API_KEY=your_api_key_here
S3_ENDPOINT=your_s3_endpoint
S3_ACCESS_KEY=your_access_key

```

## الوظائف الرئيسية

1. **رفع المستندات**: رفع صور الجواز، تذاكر الطيران، وحجوزات الفنادق
2. **استخراج البيانات**: استخدام الذكاء الاصطناعي لقراءة واستخراج البيانات تلقائياً
3. **الترجمة التلقائية**: ترجمة جميع البيانات إلى اللغة الإنجليزية
4. **تصدير PDF**: إنشاء ملف PDF جاهز للطباعة والتقديم

## البيانات المستخرجة

### بيانات الجواز
- رقم الجواز والاسم الكامل (عربي وإنجليزي)
- تاريخ الميلاد والجنسية
- تاريخ الإصدار والانتهاء
- مكان الميلاد والجنس

### بيانات الطيران
- تاريخ الذهاب والعودة
- وجهة السفر
- رقم الرحلة والحجز
- اسم شركة الطيران

### بيانات الفندق
- اسم الفندق وعنوانه (عربي وإنجليزي)
- رقم التواصل والإيميل
- تاريخ تسجيل الوصول والمغادرة
- رقم الحجز

## هيكل قاعدة البيانات

### جدول المستخدمين (users)
- معلومات المستخدم الأساسية
- طريقة تسجيل الدخول
- الصلاحيات (user/admin)

### جدول طلبات التأشيرة (visa_applications)
- بيانات الجواز
- بيانات الطيران
- بيانات الفندق
- بيانات إضافية (العنوان، رقم الهاتف، المهنة، إلخ)
- حالة الطلب (draft/completed/submitted)

## الاستضافة الموصى بها

### خيارات مجانية
1. **Railway** - استضافة كاملة مع قاعدة بيانات
2. **Render** - استضافة Backend مع PostgreSQL
3. **Vercel + Supabase** - Frontend على Vercel وقاعدة بيانات على Supabase

### خيارات مدفوعة
1. **DigitalOcean** - خادم افتراضي كامل
2. **AWS** - خدمات سحابية متكاملة
3. **Google Cloud** - بديل قوي لـ AWS
